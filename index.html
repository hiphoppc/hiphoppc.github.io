    🚀 GitHub Pages 배포 가이드

    GitHub Pages로 배포하기

    1
    GitHub에 로그인하고 오른쪽 상단 + 버튼 → New repository 클릭
    2
    Repository name을 내아이디.github.io 형태로 입력
    (예: psyco.github.io)
    Public 선택 후 Create repository 클릭
    3
    생성된 저장소 페이지에서 uploading an existing file 링크 클릭
    → 이 index.html 파일을 드래그 앤 드롭!
    → Commit changes 클릭
    4
    Settings → Pages 메뉴 → Source를 Deploy from a branch 선택
    Branch를 main / /(root) 선택 후 Save
    5
    1~2분 후 https://내아이디.github.io 에서 확인!
    끝! 🎉 이 파일 하나만 올리면 바로 볼 수 있어요.
